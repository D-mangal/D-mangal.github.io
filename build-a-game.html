Build a game

For this project we had to work in groups and make 4 different games, I decided to make the game Three in a row. The rules of the game are simple. Each player takes a turn to play a X or O and the first one to get three in a row wins. Not so hard right? While making this project I learned how to use array's to check if a player won. For this project I used HTML, CSS and Javascript.
